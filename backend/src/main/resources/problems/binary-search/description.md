
Given an array of integers `nums` sorted in ascending order, and an integer `target`, write a function to search for `target`. Return its index if found, or `-1` if not.

You must write an algorithm with `O(log n)` runtime complexity.

# Example

Input: `nums = [-1,0,3,5,9,12]`, `target = 9`
Output: `4`

Input: `nums = [-1,0,3,5,9,12]`, `target = 2`
Output: `-1`
